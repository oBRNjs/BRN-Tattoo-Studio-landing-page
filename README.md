# 🔥 BRN Tattoo Studio — Landing Page

Primeira landing page desenvolvida para um estúdio de tatuagem, com foco em design visual impactante, experiência do usuário e conversão via WhatsApp.

---

## 📌 Sobre o projeto

Este projeto foi criado como estudo prático de desenvolvimento web (HTML + CSS + JS), com o objetivo de simular uma landing page profissional para um estúdio de tatuagem.

A ideia foi ir além do básico e desenvolver algo com identidade forte, estética moderna e elementos visuais que transmitam a vibe do universo da tatuagem.

---

## 🎯 Objetivos

- Praticar estruturação semântica com HTML
- Evoluir no uso de CSS avançado (layout, animações e responsividade)
- Criar uma interface com alto impacto visual
- Simular um projeto real para portfólio
- Implementar foco em conversão (CTA para WhatsApp)

---

## 🧠 Conceitos aplicados

- Responsividade (mobile first)
- Flexbox e Grid Layout
- Animações com CSS (`@keyframes`)
- Efeitos visuais (gradientes, sombras, blur, glow)
- Scroll suave (`scroll-behavior`)
- Organização de seções (hero, serviços, galeria, depoimentos)
- UX/UI focado em conversão
- Estrutura de landing page

---

## 🎨 Design

O design segue uma linha:

- 🔥 Dark theme (preto + laranja)
- 🔥 Estética “tattoo studio”
- 🔥 Tipografia forte (Bebas Neue)
- 🔥 Efeitos de glow e partículas
- 🔥 Interface minimalista e moderna

---

## 🚀 Funcionalidades

- Hero section com impacto visual
- Navegação suave entre seções
- Carrossel de galeria estilo “Netflix”
- Botões CTA com foco em conversão
- Integração com WhatsApp
- Animações ao scroll
- Layout responsivo
